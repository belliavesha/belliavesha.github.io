What is a Mandelbrot set? Everyone knows that, it is a set of such complex numbers c, such that the process, starting from number 0 and applying and reapplying the transformation $z^2 + c$ never diverges to infinity, stays confined within a circle around the origin. It is a cool fractal. The said process can also stay confined if it is started with some number, $z_0$. The sed of such numbers $z_0$ is called a Julia set for some parameter c. They are also cool fractals. Moreove, the Mandelbrot set is the set of parameters c, such that the Julia sets are connected. Everyone knows that.

Let's take a number from the Mandelbrot set and appreciate the fact that it parametrizes a function that is applied to a variable which always stays in sight. What happens to the variable? Everyone knows that, if it is in the main cardioid, the process has a limit. Which is boring. If it is not, then it starts oscilating between n 'states', however, we can pick every n'th number from this sequence and then they are going to have a limit. Finally, if the number is exactly on the border of the set, then the process is chaotic and goes all around other points from the exact border of the set. Which is cool. But I still declare this to be rather boring, because everyone knows that.
